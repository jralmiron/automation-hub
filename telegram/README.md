# Telegram

Automatizaciones para enviar mensajes mediante `TELEGRAM_BOT_TOKEN` y `TELEGRAM_CHAT_ID`.
Útil para notificaciones diarias, avisos manuales y eventos disparados por GitHub Actions o Trigger.dev.

## Bridge de comandos

Ahora el proyecto también soporta un puente de entrada:

`Telegram -> DeepSeek -> whitelist -> GitHub Actions`

El bot puede entender frases en lenguaje natural, pero solo convertirlas en acciones seguras permitidas.
